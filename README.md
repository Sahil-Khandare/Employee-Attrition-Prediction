# Employee Attrition Prediction using Machine Learning

## Project Overview

This project predicts employee attrition using Machine Learning techniques. The objective is to identify employees who are likely to leave the organization and analyze the factors contributing to attrition.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Jupyter Notebook

## Features

* Data preprocessing and cleaning
* Exploratory data analysis
* One-hot encoding of categorical features
* Logistic Regression model implementation
* Model evaluation and accuracy analysis
* Feature importance visualization

## Dataset Features

* Satisfaction Level
* Last Evaluation
* Number of Projects
* Average Monthly Hours
* Time Spent at Company
* Work Accident
* Promotion in Last 5 Years
* Department
* Salary Level

## Machine Learning Workflow

1. Data Loading
2. Data Preprocessing
3. Feature Engineering
4. Train-Test Split
5. Logistic Regression Model Training
6. Model Evaluation
7. Feature Importance Analysis

## Results

The model successfully predicts employee attrition patterns and identifies the key factors influencing employee turnover.
